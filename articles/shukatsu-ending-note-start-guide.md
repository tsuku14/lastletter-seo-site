## エンディングノートとは何か

エンディングノートとは、自分の人生の終わりに向けて、医療・介護の希望や財産情報、家族へのメッセージなどをまとめておくための記録帳です。「終活ノート」「もしもノート」とも呼ばれます。

法律上の効力を持つ**遺言書とは異なり**、書式・内容ともに自由です。自分のペースで書き始め、何度でも書き直せるのが大きなメリットです。50代・60代はもちろん、30代・40代でも万が一の備えとして作成する方が増えています。

## 遺言書とエンディングノートの違い

| 比較項目 | エンディングノート | 遺言書 |
|---------|--------------|-------|
| 法的効力 | なし | あり（要件を満たした場合） |
| 書式 | 自由 | 法律で厳格に規定 |
| 書き直し | いつでも可能 | 改ざん防止のルールあり |
| 内容 | 医療・介護・気持ちなど幅広い | 主に財産の分配 |
| 費用 | ほぼ無料 | 公正証書の場合は数万円 |

財産の分配を確実に実現させたい場合は遺言書が必要ですが、自分の思いや情報を家族に伝える手段としてはエンディングノートが手軽で有効です。両方を作成することが理想的です。

## エンディングノートに書くべき15項目

### 【基本情報・個人情報】

**1. 個人情報**
氏名・生年月日・住所・本籍地・マイナンバーカードの保管場所。緊急時に家族が迷わないよう正確に記載します。

**2. 家族・親族リスト**
配偶者・子・兄弟姉妹・甥姪など、連絡が必要な親族の氏名・住所・電話番号を一覧化します。

**3. 自分史・略歴**
出身地・学歴・職歴・趣味など。家族が葬儀の挨拶文を作る際や遺品整理の際に参考になります。

### 【医療・介護の希望】

**4. 病気・アレルギー情報**
既往症・現在の持病・服用中の薬・血液型・アレルギー。救急搬送時に役立ちます。

**5. かかりつけ医・保険証情報**
医療機関名・担当医・健康保険証や介護保険証の保管場所を記録します。

**6. 延命治療の希望**
意識不明になった場合に延命治療を望むか否か。胃ろう・人工呼吸器の使用意向も含めて明記します。

**7. 介護の希望**
在宅介護か施設介護か、希望するケアのスタイルを書いておきます。

### 【財産・資産情報】

**8. 預貯金・金融機関リスト**
銀行名・支店名・口座番号（通帳の保管場所）。暗証番号は別の安全な場所に保管します。

**9. 不動産情報**
所有する土地・建物の所在地・権利証（登記識別情報）の保管場所。ローン残高がある場合も記載します。

**10. 保険の一覧**
生命保険・医療保険・損害保険の会社名・証券番号・担当者連絡先をまとめます。

**11. 負債・ローン情報**
住宅ローン・カードローン・連帯保証債務なども正直に記録します。家族が相続放棄を検討する材料になります。

### 【デジタル情報】

**12. デジタルアカウント・パスワード**
スマートフォンのロック解除番号・メールアカウント・SNS・クラウドストレージのID。死後のアカウント削除依頼に必要です。

### 【葬儀・お墓の希望】

**13. 葬儀の希望**
家族葬・一般葬・直葬など形式の希望、宗教・宗派、参列してほしい人のリスト、花や音楽の希望など。

**14. お墓・埋葬の希望**
既存の墓への埋葬・樹木葬・散骨・永代供養墓など希望を書きます。

### 【家族へのメッセージ】

**15. 家族へのメッセージ**
配偶者・子ども・友人など大切な人への感謝の言葉。これがエンディングノートの中でもっとも読まれる部分です。

## 書き方の3つのコツ

### 1. 「完璧」を目指さず少しずつ書く

一度に全項目を埋めようとすると挫折します。まず「名前・家族リスト・葬儀の希望」の3項目だけ書くことから始めましょう。

### 2. 定期的に見直す

財産状況や気持ちは変わります。誕生日や年末など年1回のタイミングで内容を更新する習慣をつけましょう。

### 3. 保管場所を家族に伝える

書いても誰も知らなければ意味がありません。「タンスの引き出しにある」など保管場所を家族に口頭で伝えておきましょう。

## おすすめのエンディングノート

| 種類 | 特徴 |
|------|------|
| 書店・文具店で購入できるもの | 項目がわかりやすく初心者向け。1,000〜2,000円程度 |
| 自治体が配布するもの | 無料で入手できる場合あり。市区町村の窓口へ |
| デジタル版（アプリ・PDF） | スマホで管理可能。バックアップが重要 |
| 手製（ノート・バインダー） | 自分でカスタマイズ可能。追記・差し替えが容易 |

### Q1: エンディングノートは誰に見せるべきですか？

**A1:** 原則として、もっとも信頼できる家族（配偶者・長子など）に保管場所を伝えておけば十分です。全員に見せる必要はありません。ただし、医療の希望（延命治療の可否など）については、あらかじめ家族と話し合って共有しておくことが大切です。見せる相手が複数いる場合は、「家族へのメッセージ」は個別のページや封筒に分けて管理する方法も有効です。

### Q2: エンディングノートに法的効力はありますか？財産分配を書いても有効ですか？

**A2:** エンディングノートには法的効力はありません。「〇〇に自宅を譲る」と書いても、遺言書のように法的拘束力は生じません。財産の分配を確実に実現させたい場合は、必ず別途、法的要件を満たした遺言書（自筆証書遺言・公正証書遺言）を作成してください。エンディングノートは家族への「お願い」や「気持ち」を伝える補完的なツールとして活用しましょう。

### Q3: 40代でもエンディングノートを書くべきですか？早すぎますか？

**A3:** 早すぎることはありません。むしろ40代は健康で判断力が高いうちに書ける絶好のタイミングです。急な事故や病気で意思表示ができなくなるリスクは年齢を問わず存在します。40代で作成しておけば、医療の希望や保険情報などを家族に伝えられます。定期的に見直すことで内容が充実していきます。「書き始めること」自体が目的であり、完成を焦る必要はありません。
