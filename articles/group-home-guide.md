## グループホームとは

グループホーム（認知症対応型共同生活介護）は、認知症の診断を受けた高齢者が少人数（5〜9人）で共同生活を送りながら介護を受ける施設です。「認知症の人が住み慣れた地域で家庭的な雰囲気の中で暮らす」という考え方に基づいています。

介護保険の「地域密着型サービス」に分類されるため、原則として**施設と同じ市区町村に住民票がある方のみ**入居できます。

## グループホームの特徴

- **少人数制（1ユニット5〜9人）**: アットホームな雰囲気で個別ケアが受けやすい
- **認知症専門**: スタッフが認知症ケアに特化している
- **共同生活**: 料理・掃除などの家事を一緒に行い、生活能力の維持を図る
- **地域密着型**: 同じ市区町村に住民票が必要

## 入居条件

| 条件 | 内容 |
|------|------|
| 年齢 | 65歳以上（40〜64歳は特定疾病による要介護認定が必要） |
| 要介護度 | 要支援2または要介護1〜5 |
| 診断 | 医師から認知症の診断を受けていること |
| 住民票 | 施設と同じ市区町村にあること |
| 身元保証人 | 原則として身元保証人が必要 |

※要支援1の方は入居不可。要支援2から入居可能です。

## 月額費用の目安

### 費用内訳

| 費用項目 | 月額目安 |
|---------|---------|
| 家賃・管理費 | 40,000〜80,000円 |
| 食費 | 40,000〜50,000円 |
| 介護サービス費（1割負担） | 8,000〜10,000円 |
| 日常生活費（日用品等） | 5,000〜10,000円 |
| **合計目安** | **約100,000〜150,000円/月** |

※入居一時金（0〜100万円程度）が必要な施設もあります。

### 地域による差

グループホームの費用は地域によって大きく異なります。東京都心部では月15〜20万円、地方では月10〜12万円程度が相場です。

## グループホームと特別養護老人ホーム（特養）の違い

| 比較項目 | グループホーム | 特養 |
|---------|-------------|------|
| 対象 | 認知症の方（要支援2以上） | 要介護3以上（原則） |
| 入居のしやすさ | 比較的入居しやすい | 待機待ちが多い |
| 費用 | 月10〜15万円 | 月6〜10万円（低所得者減額あり） |
| 環境 | 家庭的・少人数 | 集団的・大規模 |
| 地域 | 同一市区町村のみ | 制限なし |

## グループホームのメリット・デメリット

### メリット

- 少人数でアットホームな生活ができる
- 認知症の進行に合わせた専門的なケアを受けられる
- 家事参加で生活機能を維持しやすい
- 特養より入居待ちが少ない傾向

### デメリット

- 医療対応が限られる（看護師不在の夜間が多い）
- 重度化すると退去を求められる場合がある
- 特養より費用が高い場合が多い
- 同じ市区町村内の施設しか選べない

## グループホームの選び方5つのポイント

1. **体験入居を活用する**: 実際の雰囲気を確認。多くの施設で1〜2泊の体験入居が可能
2. **スタッフの認知症研修受講状況を確認**: 認知症介護実践者研修などの受講率が高い施設を選ぶ
3. **第三者評価の結果を確認**: 都道府県の介護サービス情報公表システムで閲覧可能
4. **医療連携の体制を確認**: かかりつけ医との連携、看取りへの対応方針を確認
5. **退去条件を必ず確認**: 状態が重度化した際の方針（転居先の紹介があるか等）

## 費用を抑えるための制度

### 高額介護サービス費

月の自己負担額が上限を超えた場合に払い戻しが受けられます。一般的な所得の方は月44,400円が上限。

### 補足給付（特定入所者介護サービス費）

住民税非課税世帯の場合、食費・居住費（家賃）が軽減されます。市区町村への申請が必要です。

### Q1: グループホームへの入居待ちはありますか？

**A1:** 特養と比べると待機期間は短い傾向にありますが、人気施設では数ヶ月〜1年以上待つ場合もあります。複数の施設に並行して申し込むことをおすすめします。地域の地域包括支援センターや無料の施設紹介サービスに相談すると、空きのある施設を早く見つけられます。

### Q2: グループホームで看取りはできますか？

**A2:** 施設によって対応が異なります。近年は「看取り加算」の算定施設が増えており、人生の最期まで同じ環境で過ごせるグループホームも増えています。入居前に「看取り対応の方針」と「ターミナルケアの体制」を確認することを強くおすすめします。

### Q3: 認知症が進行してグループホームを退去しなければならない場合はどうなりますか？

**A3:** 身体的な医療ニーズが高まり施設での対応が難しくなった場合（BPSD（行動・心理症状）が著しく重度化、医療処置が常時必要など）は転居を求められることがあります。この場合、施設側が転居先（特養・療養型施設・医療機関等）の探索を支援してくれる場合がほとんどです。入居前に「退去となる条件」と「サポート体制」を確認しておきましょう。
