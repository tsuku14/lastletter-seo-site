## 相続税とは？すべての人が払うわけではありません

相続税とは、亡くなった方から財産を受け継いだ場合にかかる税金のことです。しかし、財産を相続したすべての人に課税されるわけではありません。「遺産の総額」が、法律で定められた非課税枠である「基礎控除額」を上回った場合にのみ、申告と納税の義務が発生します。

「うちは特に裕福ではないから相続税は関係ない」と思われている方も、不動産などを含めると基礎控除額を超えるケースは少なくありません。

## 最重要！相続税の「基礎控除」とは？

相続税がかかるかどうかを判断する上で、最も重要なのが「基礎控除」です。この金額を計算し、遺産の総額がそれ以下であれば、相続税はかかりません。

### 基礎控除の計算式

基礎控除額は、以下の計算式で算出されます。

**3,000万円 ＋ (600万円 × 法定相続人の数)**

### 法定相続人とは誰のこと？数え方を解説

「法定相続人」とは、法律で定められた相続人のことです。

*   亡くなった方の配偶者（夫または妻）は、常に法定相続人になります。
*   配偶者以外は、①子、②親、③兄弟姉妹の順で優先順位が決まっています。

**例：夫が亡くなり、妻と子2人がいる場合**
この場合の法定相続人は「妻、子、子」の合計**3人**となります。

### 計算してみよう！我が家の基礎控除額

上記の例で基礎控除額を計算してみましょう。

`3,000万円 + (600万円 × 3人) = 4,800万円`

この家族の場合、遺産の総額が**4,800万円以下**であれば、相続税はかからず、税務署への申告も不要です。

## 遺産の総額はどう計算する？

遺産の総額は、預貯金や不動産、株式といった「プラスの財産」から、借金やローンなどの「マイナスの財産」を差し引いて計算します。土地や建物の評価額の計算は専門的な知識が必要になるため、正確な金額を知りたい場合は専門家の助けが必要です。

## もし基礎控除を超えたら？相続税の申告と税率

遺産の総額が基礎控除額を上回った場合は、**相続の開始があったことを知った日の翌日から10ヶ月以内**に、税務署へ相続税の申告と納税を行う必要があります。

相続税の税率は、取得する財産額が大きくなるほど高くなる「超過累進課税」が採用されています。

| 法定相続分に応ずる取得金額 | 税率   | 控除額     |
| -------------------------- | ------ | ---------- |
| 1,000万円以下              | 10%    | -          |
| 3,000万円以下              | 15%    | 50万円     |
| 5,000万円以下              | 20%    | 200万円    |
| 1億円以下                  | 30%    | 700万円    |
| ...                        | ...    | ...        |

（国税庁の速算表を基に簡略化）

## まとめ：まずは「基礎控除」の計算から始めよう

ご自身の家庭で相続税の心配があるかどうかは、まずこの「基礎控除額」を計算してみることから始まります。その上で、大まかな遺産の総額を把握することが第一歩です。

正確な財産評価や申告書の作成には、税理士などの専門家の力が必要不可欠です。もし基礎控除額を超える可能性がある場合は、早めに専門家に相談することをお勧めします。
