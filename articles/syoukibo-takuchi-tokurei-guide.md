## 小規模宅地等の特例とは

小規模宅地等の特例は、自宅や事業用の土地を相続した場合に、**その土地の評価額を最大80%減額**できる相続税の優遇制度です。

土地は相続財産の中でも評価額が高く、これなしでは自宅を手放さなければならない事態も起こりえます。この特例を正しく活用することで、相続税を大幅に軽減できます。

## 特例の種類と要件

### 1. 特定居住用宅地等（自宅の土地）

| 項目 | 内容 |
|------|------|
| 減額割合 | **80%** |
| 対象面積 | 330㎡まで |
| 評価額の効果 | 例：1億円の土地 → 2,000万円で評価 |

**適用要件（いずれかに該当）**:

**① 配偶者が相続する場合**: 取得・居住条件なし。無条件で適用可

**② 同居親族が相続する場合**:
- 相続開始前から同居していること
- 申告期限まで引き続きその自宅に居住・保有していること

**③ 家なき子特例（別居の親族）**:
- 相続開始前3年以内に本人・配偶者・3親等内の親族・同族会社が所有する家屋に住んでいないこと
- 相続開始時に居住していた家屋を過去に所有していたことがないこと
- 申告期限まで宅地を保有し続けること

### 2. 特定事業用宅地等（自営業の事業地）

| 項目 | 内容 |
|------|------|
| 減額割合 | **80%** |
| 対象面積 | 400㎡まで |
| 対象 | 相続人が事業を継続する場合 |

**要件**: 相続開始前から事業用として使用され、申告期限まで事業継続・保有すること

### 3. 特定同族会社事業用宅地等

| 項目 | 内容 |
|------|------|
| 減額割合 | **80%** |
| 対象面積 | 400㎡まで |
| 対象 | 相続人が同族会社に貸している土地 |

### 4. 貸付事業用宅地等（アパート・駐車場等）

| 項目 | 内容 |
|------|------|
| 減額割合 | **50%** |
| 対象面積 | 200㎡まで |
| 対象 | 第三者に賃貸している土地 |

## 複数の宅地がある場合の限度面積

複数の宅地を相続した場合、組み合わせによって以下の上限があります：

| 組み合わせ | 面積上限 |
|----------|---------|
| 居住用のみ | 330㎡ |
| 事業用のみ | 400㎡ |
| 居住用 + 事業用 | 730㎡（それぞれの上限を合算） |
| 居住用 + 貸付用 | 面積の按分計算が必要 |

## 注意：特例が使えないケース

### 申告前に売却・転用した場合

申告期限（10ヶ月以内）までに、対象の宅地を売却したり用途変更（自宅→賃貸等）すると特例が使えなくなります。

### 二世帯住宅で非適用になるケース

内部で行き来できない完全分離型の二世帯住宅は、2014年以降は同居とみなされるようになりましたが、要件確認が必要です。

### 相続放棄した人が取得した場合

相続放棄した人は法律上「相続人でない」ため、遺贈で受け取った場合でも適用不可です（ただし相続税は課税される）。

## 申告手続き

1. **相続税申告書**に小規模宅地等の特例の計算明細書（第11・11の2表）を添付
2. **必要書類**:
   - 登記事項証明書（土地・建物）
   - 住民票（被相続人・取得者）
   - 同居を証明する書類（必要な場合）
   - 賃貸借契約書（貸付用の場合）
3. **申告期限**: 相続開始から10ヶ月以内

**重要**: この特例は**申告書の提出が必須条件**です。申告しなければ自動的には適用されません。

## 税理士への相談が重要な理由

小規模宅地等の特例は、適用の可否・最適な組み合わせ・家なき子特例の要件など判断が複雑です。相続税に詳しい税理士への相談が強く推奨されます。専門家に依頼することで、合法的な最大限の節税が可能になります。

### Q1: 相続した自宅に住み始めるのが申告期限後になった場合、特例は使えますか？

**A1:** 特定居住用宅地等の特例（同居親族取得）では、「申告期限まで引き続きその宅地に居住していること」が要件です。申告期限（10ヶ月以内）までに居住を開始していれば問題ありません。ただし、相続開始時点で同居していたことが原則的な要件ですので、相続開始時に別居していた場合は「家なき子特例」の要件を確認してください。

### Q2: 親が老人ホームに入居中に亡くなった場合、自宅土地に特例は使えますか？

**A2:** 一定の要件を満たせば使えます。親が「要介護認定または要支援認定を受けていた」かつ「老人ホームへの入居前まで自宅に居住していた」かつ「自宅を他の人の居住用や事業用に供していない」場合、老人ホームへの入居後も自宅の土地は「特定居住用宅地等」として扱われます。これを「空き家の特例の前提」と呼びます。詳細は税理士に確認してください。

### Q3: 相続した土地を売りたい場合、いつまで保有すれば特例を使えますか？

**A3:** 申告期限（相続開始から10ヶ月）まで保有・居住を継続することが要件です。申告期限を過ぎれば売却しても特例の取り消しはありません。ただし、売却時には相続した土地の「取得費」の計算に注意が必要です。相続税申告で「取得費加算の特例」を利用すると、売却時の譲渡所得税を軽減できる場合があります。
